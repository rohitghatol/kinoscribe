Usage - Import polymer & KinoScribe

<!DOCTYPE html>
<html>
<head>
  <script
    src="...../0.3.3/platform.js"></script>
  <link rel="import"
    href=...../kinoscribe/kinoscribe.html">
    ....
</head>
  ....
<html>

Usage - Define CSS for Slide

<head>
  ....
  <style>
    .slide{
        width:600px;
        height:500px;
        border-radius: 20px;
        border:1px solid darkgray;
        box-shadow:5px 5px 5px gray;
    }
    </style>
</head>

Usage - Use the tags

<body>
  <kino-pres>
    <kino-slide x="100" y="0" z="100"
                rotateX="0" rotateY="0" rotateZ="0"
                scale="1" >
        <section class="slide">
            Slide 1
        </section>
    </kino-slide>
    .... More such slides
  </kino-pres>
</body>
